「用遊戲設計的方法激勵我的團隊」
「用第一性原理分析我的產品策略」

系統會自動匹配對應的理論智靈，注入專業思維到 AI 的回應中。

## 核心概念

### 什麼是「智靈」(Agent)？

智靈是一個封裝了特定領域專業思維的 AI Agent，包含：

- 🧠 **專業理論**：如敏捷開發、方法演技、遊戲設計
- 🎯 **使用情境**：何時該召喚這個智靈
- 💭 **思維模式**：決策風格、問題解決方法
- 🛠️ **工具權限**：可以執行的操作（讀檔案、搜尋、執行命令等）

### 複合理論混搭

靈閣最強大的地方在於「複合理論」：

```
單一理論   → 單一視角
雙重理論   → 跨界碰撞
三重理論   → 創新突破
```

**範例：**
```
「敏捷開發」→ 軟體專案管理
「敏捷開發」×「婚禮規劃」→ 用 Sprint 規劃婚禮
「第一性原理」×「心理學」×「遊戲開發」→ 從零重構遊戲循環
```

## 常見問題

### Q: 靈閣和直接問 ChatGPT 有什麼不同？

**直接問 ChatGPT：**
```
「幫我設計一個激勵系統」
→ 得到通用的建議
```

**透過靈閣召喚智靈：**
```
「用遊戲設計的思維幫我設計激勵系統」
→ 系統注入遊戲設計專業理論
→ 得到：成就系統、進度回饋、隨機獎勵、心流體驗等專業建議
```

### Q: 免費額度用完後怎麼辦？

- 可以透過 PayPal 儲值購買「魂額度」
- 每次召喚 $0.001 USD
- 額度永久有效，無使用期限

### Q: 我可以上傳自己的智靈嗎？

可以！前往 Dashboard → 上傳智靈：

1. 定義專業領域與理論
2. 設定觸發情境
3. 配置思維模式（透過問卷）
4. 授權工具權限

每次被召喚，你都能獲得 90% 分潤。

### Q: OAuth 認證安全嗎？

靈閣使用業界標準的 OAuth 2.0 認證：

- ✅ 不儲存你的 ChatGPT/Claude 密碼
- ✅ 使用 TOTP 雙重認證保護帳號
- ✅ Token 加密儲存
- ✅ HTTPS 全程加密通訊
