## 基本概念

### Q: 靈閣是什麼？

靈閣是一個透過 MCP 協議提供 **700+ 跨領域理論混搭** 的思維增強服務。

**核心價值：** 不是「找專家問答案」，而是「用 A 領域的方法解 B 領域的題」。

**範例：**
- 用「敏捷開發」規劃婚禮
- 用「方法演技」設計產品
- 用「遊戲設計」激勵團隊
- 用「第一性原理」× 心理學 × 遊戲開發

### Q: 靈閣和直接問 ChatGPT 有什麼不同？

**直接問 ChatGPT：**
```
「幫我設計一個激勵系統」
→ 得到通用的建議
```

**透過靈閣召喚智靈：**
```
「用遊戲設計的思維幫我設計激勵系統」
→ 系統注入遊戲設計專業理論
→ 得到：成就系統、進度回饋、隨機獎勵、心流體驗等專業建議
```

靈閣會自動匹配對應的專業理論，注入到 AI 的回應中，讓你得到更深入、更專業的答案。

### Q: 什麼是「複合理論混搭」？

靈閣的核心玩法是將不同領域的理論組合使用：

```
單一理論   → 單一視角
雙重理論   → 跨界碰撞
三重理論   → 創新突破
```

**700+ 理論可組合出 489,300+ 種雙理論組合！**

## 註冊與帳號

### Q: 如何註冊靈閣？

1. 前往 [https://lingge.app](https://lingge.app)
2. 輸入 Email
3. 收取驗證碼
4. 設定 TOTP 雙重認證（需要驗證器 App，如 Google Authenticator）
5. 完成註冊，自動獲得 10 次免費額度

### Q: 註冊需要付費嗎？

不需要！註冊完全免費，並且會贈送 **10 次免費召喚額度**（緣額度）。

### Q: 什麼是 TOTP 雙重認證？

TOTP (Time-based One-Time Password) 是一種雙重認證機制，透過驗證器 App（如 Google Authenticator、Authy）產生 6 位數驗證碼。

**優點：**
- ✅ 即使密碼洩漏，駭客也無法登入
- ✅ 驗證碼每 30 秒更新
- ✅ 不需要網路連線也能產生驗證碼

### Q: 忘記 TOTP 驗證器怎麼辦？

在登入頁面點擊「忘記/重設驗證器？」，透過 Email 驗證碼重新設定。

## 額度與計費

### Q: 「緣額度」和「魂額度」有什麼不同？

| 額度類型 | 取得方式 | 可召喚智靈 | 消耗金額 |
|---------|---------|-----------|---------|
| **緣額度** | 註冊贈送 | 僅限官方智靈 | 免費 |
| **魂額度** | PayPal 儲值 | 所有智靈（含用戶上傳）| $0.1/次 |

### Q: 計費透明嗎？

完全透明！你可以在 Dashboard 查看：

- 每次召喚記錄
- 消耗的額度
- 剩餘額度
- 儲值記錄

**公開承諾：** 所有計費資訊都會透過 API 驗證，確保文件與實際運作一致。

詳見 [SDD.md 的計費章節](./SDD.md#86-付費模式儲值制)。

### Q: 額度會過期嗎？

**不會！** 儲值的魂額度永久有效，沒有使用期限。

### Q: 可以退款嗎？

除法律要求外，已購買的額度不予退款。購買前請確認需求。

## 創作者與分潤

### Q: 我可以上傳自己的智靈嗎？

可以！前往 Dashboard → 上傳智靈：

1. 定義專業領域與理論
2. 設定觸發情境
3. 配置思維模式（透過問卷）
4. 授權工具權限
5. 上傳並等待審核

### Q: 上傳智靈有什麼好處？

**被動收入：** 每次你的智靈被召喚，你都能獲得 **70%** 分潤。

**範例：**
```
你的智靈被召喚 10,000 次/月
= 10,000 × $0.1 × 70%
= $700/月
```

### Q: 分潤如何發放？

- **時間**：每月 10 日自動發放（上月累積）
- **方式**：透過 PayPal 轉帳
- **門檻**：累積滿 $10 USD 才會啟動發放
- **比例**：創作者 70% / 平台 30%

### Q: 如何綁定 PayPal？

前往 Dashboard → 分潤設定：

1. 輸入 PayPal Email
2. 輸入 TOTP 驗證碼
3. 完成綁定

### Q: 分潤比例會改變嗎？

目前是 **創作者 70% / 平台 30%**。

如果未來需要調整，會：
1. 提前 30 天公告
2. 透過 Email 通知所有創作者
3. 已綁定的智靈維持原比例（僅影響新上傳智靈）

## 安全與隱私

### Q: OAuth 認證安全嗎？

非常安全！靈閣使用業界標準的 OAuth 2.0 協議：

- ✅ 不洩漏密碼給第三方應用
- ✅ Token 加密儲存
- ✅ 定期過期機制
- ✅ 可隨時撤銷授權
- ✅ HTTPS 全程加密通訊

詳細說明：[OAUTH_FLOW.md](./OAUTH_FLOW.md)

### Q: 我的資料會被洩漏嗎？

不會。靈閣遵守嚴格的資料保護政策：

- ✅ 密碼使用單向雜湊儲存（無法還原）
- ✅ TOTP 密鑰加密儲存
- ✅ 不收集或販售個人資料
- ✅ 符合 GDPR 和其他隱私法規

### Q: 召喚記錄會被公開嗎？

**不會。** 你的召喚記錄只有你自己可以看到。

**創作者可見資訊（匿名）：**
- 總召喚次數
- 收益金額

**創作者看不到：**
- 誰召喚了你的智靈
- 召喚的具體內容

## 技術與整合

### Q: 什麼是 MCP？

MCP (Model Context Protocol) 是由 Anthropic 開發的開放標準協議，讓 AI 工具能夠連接外部服務。

**官方文件：** https://modelcontextprotocol.io

### Q: MCP 設定很複雜嗎？

不複雜！只需要：

1. 在 ChatGPT/Claude 中新增 MCP Connector
2. 輸入靈閣 MCP Server URL：`https://mcp.lingge.app`
3. 選擇 OAuth 認證
4. 完成授權

詳細步驟：[MCP_SETUP.md](./MCP_SETUP.md)

### Q: 可以用 API 直接呼叫嗎？

目前靈閣僅透過 MCP 協議提供服務，沒有開放直接呼叫的 HTTP API。

### Q: 有 SDK 或函式庫嗎？

目前沒有。靈閣是為 AI 工具（ChatGPT、Claude）設計的，透過 MCP 協議整合。

如果你在開發 AI 應用並需要整合靈閣，請聯繫 support@lingge.app 討論合作可能性。

## 問題排解

### Q: 設定 MCP 後看不到任何智靈？

請檢查：

1. ✅ MCP Server URL 是否正確：`https://mcp.lingge.app`
2. ✅ OAuth 授權是否成功完成
3. ✅ 帳號額度是否充足（至少需要 1 次額度）
4. ✅ 網路連線是否正常

如果問題持續，請聯繫 support@lingge.app 並提供：
- 使用的平台（ChatGPT Desktop / Claude Code）
- 錯誤訊息截圖
- 帳號 Email

### Q: OAuth 授權失敗怎麼辦？

常見原因：

1. **瀏覽器阻擋彈出視窗**：允許彈出視窗後重試
2. **TOTP 驗證碼錯誤**：確認時間同步正確
3. **網路問題**：檢查防火牆或 VPN 設定

### Q: 召喚智靈時出現 Timeout？

可能原因：

1. 網路不穩定
2. MCP Connector 設定的 Timeout 太短
3. 靈閣伺服器繁忙

**解決方案：**
- 重試一次
- 調整 Timeout 設定（參考 [MCP_SETUP.md](./MCP_SETUP.md)）
- 如果持續發生，請回報給 support@lingge.app

### Q: 忘記密碼怎麼辦？

目前尚未提供密碼重設功能。請聯繫 support@lingge.app，我們會協助你重設帳號。
